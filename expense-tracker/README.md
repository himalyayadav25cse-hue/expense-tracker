## 💡 What does it do?

A beginner AI project that helps you:

| Feature | Description |
|--------|-------------|
| ➕ Add Expenses | Log daily spending with category & description |
| 📋 View Expenses | See all your expenses in a clean table |
| 📊 Summary | Visual breakdown of spending by category |
| 🤖 AI Tips | Smart personalized money-saving advice |
| 💾 Auto Save | All data saved automatically to your PC |

---

## 🚀 How to Run

### Requirements
- Python 3.x only — **no extra libraries needed!**

### Run the program
```bash
python expense_tracker.py
```

## 🧠 How does the AI work?

```
Your Expenses ──► AI Analysis ──► Smart Tips
   Food: 40%        Rule-based       "Cook at home!"
   Shopping: 30%    + Pattern         "Set a budget!"
   Transport: 20%   Detection         "Try carpooling!"
```

The AI uses **rule-based logic** and **pattern detection** to:
- Identify your biggest spending categories
- Calculate what percentage you spend on each
- Generate personalized tips based on your habits
- Suggest a monthly savings goal
